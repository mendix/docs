## Items

The dropdown buttons contains a list of items. Each item has the following properties.

### Action

You can set the following on-click actions to be performed when an item is clicked:

* Call microflow
* Call nanoflow 
* Open page 
* Save 
* Cancel 
* Link 
* Close page 
* Delete 
* Sync
