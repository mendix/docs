The 'Change Variable' action can be used to change the value of an existing variable.

<div class="alert alert-info">{% markdown %}

See [Microflow Element Common Properties](/refguide5/microflow-element-common-properties) for properties that all activities share (e.g. caption). This page only describes the properties specific to the action.

{% endmarkdown %}</div>

## Input Properties

### Variable name

The variable of which you want to change the value.

## Action Properties

### Value

The new value for the variable. The value can be entered using [microflow expressions](/refguide5/microflow-expressions). The type of the microflow expression should be the same as the type of the selected variable.
